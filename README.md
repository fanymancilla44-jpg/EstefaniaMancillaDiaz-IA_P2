# Práctica 2 — Métodos de Ordenamiento
Scripts individuales (C#) para: BubbleSort, InsertionSort, SelectionSort, ShellSort,
QuickSort, MergeSort y RadixSort.

**Cómo ejecutar (Windows/.NET):**
- Compilar y correr uno:
  csc BubbleSort.cs && BubbleSort.exe
- Introduce números separados por espacio y el script los ordena.

> Cada archivo incluye comentarios con el tipo de algoritmo y cómo funciona.
